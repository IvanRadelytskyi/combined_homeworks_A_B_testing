# 🚀 A/B Тестування: Від планування до аналізу результатів
> **Комплексний аналітичний кейс: Оптимізація конверсії преміум-підписки**

---

### 📌 Про проєкт
Цей проєкт об'єднує два ключові етапи роботи над маркетинговим експериментом:
* **Етап 1. Проєктування (Planning):** Розрахунок параметрів тесту на основі вхідних бізнес-даних.
* **Етап 2. Аналіз (Analysis):** Обробка результатів **21-денного тесту** за допомогою Python та прийняття фінального рішення.

**🎯 Мета:** Перевірити вплив знижки 50% на конверсію нових користувачів у купівлю преміум-підписки.

---

### 🛠️ Технологічний стек
* **Python (Pandas)** — обробка та очищення даних.
* **Matplotlib / Seaborn** — візуалізація результатів та динаміки.
* **Statsmodels** — статистична перевірка (Z-test).
* **Jupyter Notebook** — середовище розробки та розрахунків.

### 📋 Етап 1: Планування (Design)
На основі попередніх даних було сформовано дизайн тесту:
* **Гіпотеза:** Новий варіант дизайну зі знижкою збільшить конверсію.
* **Цільова вибірка:** 13,146 користувачів.
* **Розрахункова тривалість:** 7 днів (трафік 2,000 користувачів/день).
* **Основна метрика:** Install-to-Paid Conversion Rate.

---

### 📊 Етап 2: Реальний аналіз (Implementation)
Фактичний тест тривав **21 день**, що дозволило отримати стабільні дані та нівелювати ефект новизни.

**Результати тесту:**
* **Група А (Контроль):** 10,013 користувачів, конверсія **6.10%**.
* **Група В (Тест):** 9,985 користувачів, конверсія **8.90%**.
* **Результат:** Варіант В спрацював на **46% краще**, ніж контроль.
